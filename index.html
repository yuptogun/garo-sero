.garo.sero

= simple table layout helper CSS

If you speak Korean it is really easy to get how it works. Otherwise, please learn English and refer to the corresponding classnames.

One thing to mention: don't mix two languages. I mean, something like .garo.wrapper won't work.

  • .ro(.uro) = .wrapper : ~(으)로 ("-ly")
  • .garo = .horizontal : 가로 (usually known as rows)
  • .sero = .vertical : 세로 (usually known as cells)
  • .chum = .first : 처음 (either row or cell)
  • .qut = .last : 끝 (either row or cell)
  • .kan = .cell : 칸 (centeted and middled by default)
  • .oen = .lefted : 왼
  • .orn = .righted : 오른
  • .oui = .topped : 위
  • .ale = .bottomed : 아래

.garo.sero.ro. > .sero.kan
가로 세로로 > 세로칸
→ horizontally & vertically wraps one vertical section

.garo.sero.ro. > .sero.oen.kan
가로 세로로 > 세로 왼칸
→ horizontally & vertically wraps one vertical section that aligns everything left

.garo.sero.ro. > .sero.orn.ale.kan
가로 세로로 > 세로 오른아래 칸
→ horizontally & vertically wraps one vertical section that aligns everything bottom right

.garo.sero.ro. > .garo.ro
가로 세로로 > 가로로
→ this row uses minimum required height

.garo.sero.ro. > .garo.qut.uro
가로 세로로 > 가로 끝으로
→ this row extends to the horizontal & vertical end of the wrapper
(you need to add .kan inside to make it centered)
(it is identical with .garo.sero.ro > qut.garo.ro = 가로 세로로 > 끝 가로로)
